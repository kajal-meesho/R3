# KRD: SPSE-5763 — Format Indian Rupee currency amounts

## Ticket
- ID: SPSE-5763
- Title: Format Indian Rupee currency amounts
- Status: Open
- Priority: Major
- Description: We need a small reusable utility function that formats an integer amount of paise into a properly formatted Indian Rupee string.
Example use:  format_inr(12345)       →   should produce something like "₹ 123.45"  format_inr(10000000)    →   should produce something like "₹ 1,00,000.00"
Used internally by the admin dashboard to display order amounts in a consistent way. Should be importable as a standalone helper.
Edge cases worth considering:
zero amount
negative amount
very large amounts (above ₹ 1 crore)
Pick reasonable defaults but ask if anything in the spec is unclear.

## Goal
Implement a standalone helper function `formatInr(paise)` that converts an integer amount of paise into a formatted Indian Rupee string (e.g., `"₹ 123.45"`, `"₹ 1,00,000.00"`). Must handle zero, negative, and very large amounts with reasonable defaults.

## Decisions made
- **Language**: Java (chosen by @user)
- **Negative format**: `-₹ 123.45` (minus sign placed before the rupee symbol)
- **Decimal places**: always two, even for whole numbers (e.g., `₹ 100.00`)
- **Indian numbering**: last three digits grouped, then every two digits; commas used as grouping separators
- **Rupee symbol**: Unicode `₹` (`\u20B9`)

## Current proposal
```java
/**
 * Formats an amount of paise into an Indian Rupee string with Indian numbering.
 *
 * @param paise the amount in paise (integer)
 * @return formatted string like "₹ 1,00,000.00" or "-₹ 123.45" when negative
 */
public static String formatInr(long paise) {
    // Determine sign
    boolean negative = paise < 0;
    long absPaise = Math.abs(paise);

    long rupees = absPaise / 100;
    int paiseRem = (int) (absPaise % 100);

    // Format rupees with Indian grouping
    String rupeeStr = Long.toString(rupees);
    StringBuilder grouped = new StringBuilder();
    int len = rupeeStr.length();
    if (len <= 3) {
        grouped.append(rupeeStr);
    } else {
        // Last 3 digits
        grouped.append(rupeeStr, len - 3, len);
        String remaining = rupeeStr.substring(0, len - 3);
        // Groups of 2 from right
        int remLen = remaining.length();
        for (int i = remLen; i > 0; i -= 2) {
            int start = Math.max(0, i - 2);
            if (grouped.length() > 0) {
                grouped.insert(0, ',');
            }
            grouped.insert(0, remaining, start, i);
        }
    }

    // Format paise remainder (always two digits)
    String paiseStr = String.format("%02d", paiseRem);

    // Assemble final string
    String formatted = "\u20B9 " + grouped.toString() + "." + paiseStr;
    if (negative) {
        formatted = "-" + formatted;
    }
    return formatted;
}
```
